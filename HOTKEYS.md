# Keyboard Shortcuts

Quick reference for all keyboard shortcuts in the app.

## Files Page

| Shortcut | Action |
|----------|--------|
| `?` | Show this help (opens in right pane) |
| `/` | Focus sidebar search |
| `Ctrl+B` or `Alt+[` | Toggle sidebar visibility |
| `Ctrl+\` | Toggle split view |
| `Ctrl+Shift+C` or `Alt+]` | Toggle AI chat panel |
| `Ctrl+G` | Toggle git graph in right pane |
| `Ctrl+Shift+B` | Toggle beads board |
| `Ctrl+Click` | Open file in right pane (when split view active) |
| `Escape` | Clear focus |

### Sidebar Navigation

| Shortcut | Action |
|----------|--------|
| `Tab` | Move focus down the file tree |
| `Shift+Tab` | Move focus up the file tree |
| `Enter` | Open focused file |
| `Right Click` | Open context menu |

## Source Control Page

| Shortcut | Action |
|----------|--------|
| `/` | Focus search box |
| `j` | Move to next repo |
| `k` | Move to previous repo |
| `Enter` | Toggle expand/collapse focused repo |
| `r` | Refresh all repos |
| `Escape` | Clear search, or collapse all repos |

## Prompts Page

| Shortcut | Action |
|----------|--------|
| `Tab` | Move between inline fields |
| `Enter` | Save current field |

## Global

| Shortcut | Action |
|----------|--------|
| `Escape` | Close modals and popups |

---

*Press the keyboard icon in the toolbar to view this reference anytime.*
